📝 Smart-Notes-App

Smart-Notes-App is a core module for the Nexa AI Voice Assistant. It transforms simple voice or text inputs into an organized digital database, specifically designed for students and developers to manage their daily tasks and academic assignments.

✨ Key Features

• Smart Storage: Uses JSON to store notes as structured data, making them easy to search and sort.

• Auto-Timestamp: Every note is automatically tagged with the exact date and time.

• Nexa Integration: Optimized to work with Python voice recognition for hands-free note-taking.

• CRUD Operations: Full support to Create, Read, and Delete notes using unique IDs.

• Privacy First: All data is stored locally on your device.

🛠️ Technical Implementation

Built using HTML, CSS and JavaScript with localStorage, drawing canvas, and browser-based media and speech features.

📖 Usage Example

Input: "Submit BA Sem 1 Assignment by Friday"

App Action: Saves the note with timestamp for future retrieval.
